#  Casos de Uso

## UC01 - Cargar archivos Word
**Actor:** Usuario  
**Descripción:** Permite seleccionar uno o varios archivos `.docx` para su edición.  
**Resultado esperado:** Los documentos se muestran en el área de texto.

## UC02 - Buscar texto
**Actor:** Usuario  
**Descripción:** Ingresa un texto en el campo “Buscar” y resalta las coincidencias.  
**Resultado esperado:** Las coincidencias aparecen resaltadas en amarillo.

## UC03 - Reemplazar texto selectivamente
**Actor:** Usuario  
**Descripción:** Permite confirmar reemplazo uno por uno.  
**Resultado esperado:** El texto se actualiza según confirmaciones del usuario.
